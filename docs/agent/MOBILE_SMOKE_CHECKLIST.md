# 手机端一日主链路验收清单

## 使用场景

本清单用于手机端测试包人工验收，目标是确认 Buddy 在真实手机体验中可以完成“一日主链路”。

测试包构建前应确认：

```powershell
cd E:\buddy\buddy-client
npm run api:cloud
```

构建完成或恢复本地开发前应执行：

```powershell
npm run api:local
```

## 前置条件

- [ ] 手机端测试包已安装并可打开。
- [ ] 测试包连接 Cloud API：`http://101.133.130.137/api/v1`。
- [ ] Cloud Staging 后端健康检查通过。
- [ ] 测试账号使用本轮新建账号，避免历史数据干扰。
- [ ] 不使用真实儿童隐私数据。

- [ ] 新学生账号注册成功。
- [ ] 新学生账号登录成功。

- [ ] 错误密码提示为可理解文案，例如“用户名或密码错误”。
- [ ] 错误密码不会进入 Main。
